## 📊 Dataset Overview

The dashboard includes the following data dimensions:

| Dimension | Description |
|-----------|-------------|
| **Year** | 2023, 2024, 2025 |
| **Month** | January to December |
| **Product Category** | Electronics, Furniture, Home Appliances, Clothing, Sports |
| **Expenses** | Actual and Budgeted expense amounts |
| **Income** | Actual and Budgeted income amounts |
| **Variance %** | Percentage difference between actual and budget |

**Data Granularity:** Monthly records for each product category

## 📈 Dashboard Features

### Main Components:

1. **Expenses Card**
   - Shows total actual expenses
   - Displays budgeted amount
   - Calculates variance percentage
   - Circular gauge showing % of budget utilization

2. **Income Card**
   - Tracks actual income generated
   - Compares against income budget
   - Shows variance trend
   - Circular gauge for budget performance

3. **Year-over-Year Trend**
   - Bar charts showing expenses and income trends across 2023-2025
   - Helps identify growth/decline patterns

4. **Category Breakdown Table**
   - Detailed view of each product category
   - Budget vs Actual comparison
   - Red/Green indicators for quick status check

5. **Dynamic Filters**
   - Filter by Year, Month, and Product Category
   - Allows drilling down into specific periods and categories

## 💡 Key Insights

Based on the dashboard analysis:

### October 2025 (Electronics Category):
- **Expenses:** ₦5,034,498 actual vs ₦5,046,967 budget (-25% variance - Under Budget ✅)
- **Income:** ₦7,745,494 actual vs ₦7,809,989 budget (-83% variance)
- The Electronics category is performing well on expense control

### March Analysis (Full Year):
- **Expenses:** ₦69,155,964 actual vs ₦69,070,563 budget (-0.12% variance)
- **Income:** ₦111,071,503 actual vs ₦110,285,134 budget (+0.71% variance ✅)
- Overall, expenses are tightly controlled near budget
- Income is exceeding projections, which is positive

### Category Performance:
- **Sports** - Highest spending category (₦12M+ budget in March)
- **Furniture & Electronics** - Second tier spending
- **Home Appliances & Clothing** - Smaller budget allocations

### Trends Observed:
- Most categories show red indicators (expenses exceeding budget) in several months
- Income is generally tracking well against budget
- Year-over-year comparison shows variations across 2023-2025

## 🚀 How to Explore

### Opening the Dashboard:
1. Download the `.pbix` file from this repo
2. Open with **Power BI Desktop** (free download from Microsoft)
3. Interact with the filters at the top-right

### Dashboard Interaction:
- **Year/Month/Category Dropdowns** - Slice and dice data in real-time
- **Hover over visuals** - See exact values and percentages
- **Click bars** - Some visuals support drill-through (depending on your Power BI version)
- **Variance Gauge** - Shows % of budget used at a glance

### Dashboard Design Notes:
- 🔴 **Red indicators** = Actual expenses exceed budget
- 🟢 **Green indicators** = Actual income meets/exceeds budget  
- 📊 **Circular gauges** = Overall budget utilization percentage
- 📈 **Bar charts** = Year-over-year trend comparison
